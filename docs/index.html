Three Websites, Built From Scratch

Product
Python Product Development Automation Recruitment Tech

ClientWatch

Built an internal job intelligence platform that monitors external job boards in real time and alerts recruitment consultants when existing clients post roles publicly. The tool identifies a gap that traditional CRM systems miss entirely: clients quietly hiring through competitors. By surfacing these signals automatically, it turns passive client relationships into active placement opportunities before a competitor gets there first. Built independently using Python, with version control via GitHub.

Proactive placement opportunities surfaced automatically from passive client monitoring

Analytics GA4 Google Sheets JavaScript Data Viz

Live KPI Performance Dashboard

Designed and built a real-time marketing performance dashboard for Integrated Staffing, pulling live data from Google Sheets via a custom Apps Script JSON endpoint. The dashboard tracks social media KPIs, website analytics, and quarter-over-quarter trends, with animated counters, Chart.js visualisations, automatic delta calculations, and CSV export. Built to replace fragmented manual reporting and give leadership a single source of truth on marketing performance.

Replaced manual monthly reporting with a live, self-updating performance view
